What are Variables?

Variables store data.

There is two types of Variables.

Local and Global

Global variables are variables that can be accessed from anywhere in the code.

A global variable looks like this -> var global = 50;

While local variables are only accessed wthin FUNCTIONS. I will be explaining functions next.

What are Functions?

Functions are a section in a code that doesnt run, unless it is called upon!

Functions are great for making a control structure.

What are Control Structures?

Control structures are what they say they are.

Its a system to control everything and have a nice smooth flow in the code.
